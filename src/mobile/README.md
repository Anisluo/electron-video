# mobile

## 手机端项目
```
通过vue-cli手脚架编写的一个移动端项目,和客户端项目是独立的。
以客户端提供的web服务接口作为数据来源,
该项目开发完毕打包后会放入客户端根目录static下,
通过客户端设置页面中的二维码扫一扫，即可访问打包好的移动端项目。
```
